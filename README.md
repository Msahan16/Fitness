# Fitness GYM - Responsive Promotional Website

A premium, modern, and fully responsive promotional website for a local fitness brand. This project showcases the gym's services, membership plans, expert trainers, and allows interested users to submit inquiries through a validated contact form.

## 🚀 Live Demo
[Live Link Placeholder](https://fitness-gym-demo.vercel.app)

## 🛠️ Tech Stack
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Vanilla CSS Modules](https://github.com/css-modules/css-modules)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Form Handling**: [React Hook Form](https://react-hook-form.com/)
- **Validation**: [Zod](https://zod.dev/)

## ✨ Key Features
- **Modern UI/UX**: Premium dark-themed design with vibrant gold/orange accents.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop screens.
- **Smooth Animations**: Entrance animations and hover effects using Framer Motion.
- **Validated Contact Form**: Client-side validation with instant feedback on submission.
- **Sectioned Layout**: Includes Hero, About, Services, Trainers, and Contact sections.
- **SEO Optimized**: Semantic HTML and proper metadata for search engine indexing.

## ⚙️ Setup Instructions

### Prerequisites
- Node.js (v18.0.0 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/fitness-gym.git
   ```
2. Navigate to the project directory:
   ```bash
   cd fitness-gym
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build for Production
```bash
npm run build
npm run start
```
